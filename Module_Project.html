Module Project

Mini Coin Operated Snack Machine

Description

It dispenses snacks(choclate bars, sweets), using a gravity feed magazine, whenever a coin is inserted. After inserting a $1 coin, the user can select between two options through the push buttons. A servo is used to dispense the snack.

Features

  1. A PIR motion detector sensor is used to detect the coin inserted.
  2. Push buttons for users to select options.
  3. Servo to dispense snack.
Optional(If there is enough time)
  1. Only $1 coins are accepted. The coin slides through a compartment, it will fall through and get rejected if it is smaller than a 1$ coin. (only works for newer coins)
  2. A mechanism controlled by a servo to return coin if user changed their mind.
